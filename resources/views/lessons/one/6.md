# 7. <a name="getting-result"></a> Getting results

Using the data grid object, you can retrieve the resulting data by calling `toArray` or `toJson` on it.

```
$grid = DataGrid::make($handler, [
    'filters' => [
        'country' => 'US',
    ],
    'page' => 5,
]);

return $grid->toJson();
```

The response would look similar to this object.

```
{
    "total": 3,
    "filtered": 3,
    "page": 1,
    "pages": 1,
    "method": 'single',
    "threshold": 10,
    "throttle": 10,
    "filters": [],
    "sorts": [
        {
            "column": "name",
            "direction": "asc"
        }
    ],
    "perPage": 10,
    "previousPage": null,
    "nextPage": null,
    "results": [
        {
            "name": "John Doe",
            "age": 22,
            "location": "New York"
        },
        ...
    ]
}
```
